# PWA Icons Needed

Please add the following icon files to this directory:

## Required Sizes
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Recommendations
- Use the Ohmitex logo
- PNG format with transparent background
- Square dimensions (1:1 ratio)
- For maskable icons, keep safe zone (center 80%)

## Tools to Generate
You can use these tools to generate all sizes from one source:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://maskable.app/editor (for maskable icons)
